# Revolta

> AI 대화를 자동으로 수집·저장하고, 맞춤형 학습 인사이트를 제공하는 브라우저 확장 프로그램

---

## 1. 프로젝트 소개

Revolta는 ChatGPT, Claude, Gemini 등 다양한 AI 서비스와 주고받은 대화를 자동으로 수집·저장하고, 이를 기반으로 맞춤형 학습 요약과 인사이트를 제공하는 서비스입니다.

사용자는 별도의 기록 작업 없이 평소처럼 브라우저에서 AI를 사용하기만 하면, Revolta 확장 프로그램이 백그라운드에서 대화를 자동으로 수집하고 정리·분석해줍니다.

---

## 2. 해결하려는 문제

| 문제 | Revolta의 해결 방식 |
|------|----------------------|
| AI 대화 내용이 시간이 지나면 묻혀서 다시 찾기 어렵다 | 모든 대화를 자동 수집·저장하여 언제든 검색 가능 |
| 여러 AI 서비스를 혼용하면 대화가 플랫폼별로 흩어진다 | ChatGPT, Claude, Gemini 등 크로스 플랫폼 대화를 한곳에서 통합 관리 |
| 어떤 주제에 대해 질문했는지 한눈에 파악하기 힘들다 | 질문을 자동으로 태깅·분류하여 주제별로 정리 |
| 대화에서 얻은 지식을 복습하거나 정리할 시간이 부족하다 | 일일 요약 리포트를 자동 생성하여 핵심만 전달 |

---

## 3. 대상 사용자

- 학습, 업무, 리서치 등 다양한 목적으로 AI를 활발히 사용하는 모든 사용자
- 여러 AI 서비스를 혼용하면서 대화 기록이 흩어져 불편한 사용자
- 대화 기록을 따로 정리하는 시간을 줄이고 싶은 사용자
- 과거 AI 대화를 다시 찾아보고 싶은 사용자

---

## 4. MVP 핵심 기능

### 4-1. 대화 자동 수집 및 저장 (핵심)

- Chrome 확장 프로그램이 백그라운드에서 동작하며, 사용자가 AI 서비스를 이용할 때 질문과 응답을 실시간으로 자동 수집
- MVP 지원 AI 플랫폼: ChatGPT, Claude, Gemini
- 수집된 대화는 클라우드 서버에 안전하게 저장
- 대화별 메타데이터 기록: AI 플랫폼명, 날짜/시간, 대화 세션 구분, 질문 수, 응답 길이 등
- Chromium 기반 브라우저(Chrome, Edge, Brave 등)에서 모두 동작

### 4-2. 질문 자동 태깅 및 분류

- 수집된 질문을 AI가 분석하여 자동으로 주제 태그 부여 (예: `#프로그래밍`, `#영어학습`, `#마케팅`)
- 태그 기반으로 대화를 주제별로 그룹핑하여 확인 가능 (플랫폼 구분 없이 통합)
- 사용자가 태그를 수동으로 수정·추가할 수 있는 기능 제공

### 4-3. 일일 요약 리포트

- 하루 동안 나눈 대화를 요약하여 리포트 자동 생성
- 리포트 포함 내용:
  - 오늘의 주요 질문 목록
  - 주제별 대화 비중 (예: 프로그래밍 60%, 영어 30%, 기타 10%)
  - AI 플랫폼별 사용 비중 (예: ChatGPT 50%, Claude 30%, Gemini 20%)
  - 핵심 인사이트 요약 (각 대화에서 얻은 주요 답변 한 줄 정리)
- 리포트 확인 방법: 확장 프로그램 대시보드에서 날짜별 조회

### 4-4. 간단 검색 기능

- 키워드 기반으로 과거 대화를 검색
- AI 플랫폼, 태그, 날짜 범위, 키워드 조합으로 필터링
- 검색 결과에서 원문 대화로 바로 이동 가능

---

## 5. 사용자 시나리오

### 시나리오 1: 일상적인 사용

> 사용자 A는 평소 ChatGPT로 코딩 질문을, Claude로 문서 작성을 합니다. Revolta 확장 프로그램을 설치한 뒤 평소처럼 브라우저에서 AI를 사용하면, 두 플랫폼의 대화가 모두 자동으로 저장됩니다. 퇴근 전 일일 리포트를 확인하여 오늘 배운 내용을 한눈에 복습합니다.

### 시나리오 2: 크로스 플랫폼 검색

> 사용자 B는 2주 전에 API 인증에 대해 질문했지만, ChatGPT에서 물어봤는지 Claude에서 물어봤는지 기억이 나지 않습니다. Revolta에서 "API 인증"을 검색하면 플랫폼에 관계없이 관련 대화가 모두 나타나고, 바로 원문으로 이동할 수 있습니다.

### 시나리오 3: 학습 패턴 파악

> 사용자 C는 자신이 어떤 주제를 주로 질문하는지, 어떤 AI를 더 많이 사용하는지 궁금합니다. 일일 리포트의 주제별·플랫폼별 비중을 통해 자신의 AI 활용 패턴을 파악하고, 부족한 영역을 집중적으로 공부하기로 합니다.

## 6. 경쟁 서비스 분석

| 서비스 | 형태 | 핵심 기능 | Revolta 차별점 |
|--------|------|-----------|-----------------|
| Chat Memo | Chrome 확장 프로그램 | 멀티 플랫폼 대화 자동 저장, 로컬 저장, 키워드 검색 | Revolta는 클라우드 저장으로 디바이스 간 접근 가능, AI 기반 자동 태깅·일일 리포트 등 분석 기능 제공 |
| AI Exporter / SaveYourChat | Chrome 확장 프로그램 | 대화를 PDF, Markdown 등으로 내보내기 | 단순 백업/내보내기에 그침. Revolta는 저장된 데이터를 분석·가공하여 인사이트 제공 |
| 각 AI 서비스 자체 기능 | 플랫폼 내장 | 자체 대화 히스토리, 검색, 메모리 | 자기 플랫폼 대화만 관리 가능. 크로스 플랫폼 통합 관리·분석 불가 |

Revolta의 핵심 차별점은 **"AI 대화의 지식화"**에 있습니다. 경쟁 서비스 대부분이 "저장"에 머물러 있는 반면, Revolta는 저장된 대화에서 자동 태깅, 일일 요약 리포트, 크로스 플랫폼 통합 분석 등 사용자가 직접 하기 어려운 가공·분석을 제공합니다.

## 7. MVP 성공 기준

출시 후 8주(2개월) 기준으로 아래 지표를 추적합니다.

| 지표 | 목표 | 측정 의미 |
|------|------|-----------|
| 확장 프로그램 설치 수 | 500+ | 시장 관심도 검증 |
| 주간 활성 사용자 (WAU) | 설치 대비 30%+ | 실제 사용 가치 검증 |
| 평균 수집 대화 수 (일/인) | 3건+ | 핵심 기능 작동 여부 |
| 일일 리포트 열람률 | WAU 대비 40%+ | 분석 기능의 가치 검증 |
| 검색 기능 사용률 | WAU 대비 20%+ | 재방문 동기 검증 |
| 7일 리텐션 | 25%+ | 지속적 사용 가치 확인 |

핵심 판단 기준: **리텐션**과 **리포트 열람률**. 사용자가 리포트를 반복적으로 확인한다면 서비스가 실질적 가치를 제공하고 있다는 신호입니다.

## 8. MVP 제외 범위 (향후 확장 예정)

- 외부 메신저 연동 (Slack, Telegram 등을 통한 알림)
- 고급 분석 및 자동 퀴즈/복습 모드
- 다중 디바이스 간 실시간 동기화
- ChatGPT, Claude, Gemini 외 추가 AI 플랫폼 지원 (Perplexity, DeepSeek 등)

---

## 9. 제품 로드맵

### Phase 1 — MVP (현재)

- Chrome 확장 프로그램 출시 (Chromium 기반 브라우저 지원)
- ChatGPT, Claude, Gemini 대화 자동 수집 및 클라우드 저장
- 질문 자동 태깅/분류
- 일일 요약 리포트
- 키워드 및 필터 검색

### Phase 2 — 학습 강화 및 모바일 확장

- 퀴즈/복습 모드 (수집된 대화 기반 자동 생성)
- 주간/월간 리포트 및 학습 트렌드 분석
- iOS / Android 모바일 앱 출시 (대시보드·리포트 조회 중심)
- Safari Web Extension 지원으로 iOS/macOS 사용자 대화 수집 확장
- 추가 AI 플랫폼 지원 (Perplexity, DeepSeek 등)

### Phase 3 — 플랫폼 고도화

- 모바일 앱 대화 수집 기능 확장
- 외부 메신저 알림 연동
- 팀/조직 단위 대시보드 제공
- 다중 디바이스 간 실시간 동기화
