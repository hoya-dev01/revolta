코드 리뷰를 수행해줘: $ARGUMENTS

지정된 파일 또는 최근 변경된 파일을 분석하고, 아래 기준으로 리뷰해줘.

## 리뷰 기준

### 🔴 Critical (반드시 수정)
- 타입 안전성 위반 (`any` 사용, 타입 단언 남용)
- 보안 취약점 (하드코딩된 키, XSS, 데이터 유출 가능성)
- Content script에서의 안전하지 않은 DOM 접근
- 누락된 에러 핸들링 (try-catch 없는 async)

### 🟡 Warning (수정 권장)
- CLAUDE.md 컨벤션 미준수
- 불필요한 re-render 유발 코드
- 중복 코드 (shared 패키지로 추출 가능)
- 테스트 커버리지 부족

### 🟢 Suggestion (개선 제안)
- 더 나은 네이밍
- 코드 구조 개선
- 성능 최적화 기회

## 출력 형식

파일별로 그룹화하여 리뷰 결과를 정리해줘.
각 항목에 심각도 아이콘, 라인 번호, 구체적인 개선 방안을 포함해.

⚠️ 리뷰 에이전트는 읽기 전용이다. 코드를 직접 수정하지 마.
